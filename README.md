# Scoutocode

Un codeur de messages secrets pour les activités et grands jeux scouts.
Morse, Code des templiers, Décalages comme K7 ou avocat, binaire, etc.

**[Utiliser Scoutocode en ligne](https://scoutocode.github.io/scoutocode/)**

## Développement

```bash
npm install
npm run dev
```
