# Symboles Mandarin SVG

Placez ici vos fichiers SVG nommés selon les lettres de l'alphabet :

- `a.svg` pour la lettre A
- `b.svg` pour la lettre B
- `c.svg` pour la lettre C
- ...
- `z.svg` pour la lettre Z

## Format attendu

- Dimensions recommandées : 48x48 pixels
- Format : SVG valide
- Nom du fichier : lettre minuscule + `.svg`

## Fichier d'exemple

Le fichier `a.svg` est un exemple placeholder. Remplacez-le par votre propre symbole mandarin pour la lettre A, et ajoutez les fichiers pour les autres lettres (b.svg, c.svg, etc.).
